# NoSE_Index

## インストール手順

```
git clone https://github.com/Y-Wakuta/NoSE_Index.git
cd NoSE_Index/nose-cli-master
bundle install --without=development mysql
```

これを実行した後，以下を実行する．

```
sudo apt install coinor-libcbc3 \
                       coinor-libcbc-dev \
                       coinor-libcgl-dev \
                       coinor-libclp-dev \
                       coinor-libcoinutils-dev \
                       coinor-libosi-dev \
```
